## Project Overview

This project was designed as a personal analytics and monitoring tool connected to an Excel dataset where betting activity is continuously recorded and updated.

The dashboard helps track performance, analyze profitability, monitor bankroll evolution, and identify trends over time through data-driven insights.

---

## Main Features

* ROI analysis
* Profit & loss tracking
* Win rate monitoring
* Bankroll evolution
* Interactive filters
* Performance trends over time
* Analysis by sport, market, and competition
* Betting volume monitoring
* Odds interval analysis


---

## Purpose

The objective of this dashboard is to maintain structured financial control, improve decision-making through analytics, and provide transparent tracking of betting performance over time.
